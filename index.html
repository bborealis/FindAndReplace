Find and Replace

Find and replace a word 09/11/2015

By Brian Borealis

Description

User enters a string and then chooses a word to find and replace with another word. User is able to find and replace multiple cases of the same word.

Setup

Clone and open index.html in your browser.

Technologies Used

This project makes use of: Mocha and Chai as testing-frameworks and has a dependency for jQuery.
